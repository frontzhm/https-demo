# enhanced-table

## 先安装依赖

```
yarn install
```

### 本地启动

```
yarn run serve
```

### 打包

```
yarn run build
```

运行`node server.js`之后打开
[http://localhost:8080/single-query](http://localhost:8080/single-query)即可看到效果
